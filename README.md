# Virtual Office 项目

一个前端基于 Electron + React + TypeScript / 后端基于 Spring Boot 的虚拟办公空间应用，提供在线协作、实时聊天和视频会议等功能。

## 项目概览

Virtual Office 是一个现代化的远程办公协作平台，通过虚拟空间模拟线下办公室环境，让团队成员可以在线进行实时交流和协作。

## 操作流程

打开应用进入载入页面：显示应用logo和应用名
进入登陆注册页面：用户可以通过登陆注册页面进行账号认证（登陆：用户输入邮箱+密码，后端验证后返回token； 注册：用户输入账号密码、用户名、邮箱等信息，后端验证后创建新用户账号）
注册时可以选择身份，分为普通用户和管理员用户 （普通用户：默认注册身份；管理员用户：需要在注册时提供管理员权限，用于管理团队成员和虚拟办公空间）
进入主页面：侧边栏显示虚拟办公空间、聊天频道、视频会议入口和用户形象设置，用户可以通过点击侧边栏图标切换不同功能模块
虚拟办公空间：用户可以在虚拟办公空间中查看其他成员的位置和状态，同时可以通过拖动用户形象来调整自己的位置也可以设置自己的状态，例如咖啡时间头像变成咖啡杯logo，会议时间头像变成视频会议logo等
聊天频道：用户可以加入公共聊天频道或创建私人聊天频道，与团队成员进行实时交流（提供翻译功能）
视频会议：用户可以创建视频会议房间，邀请其他成员加入，进行实时音视频通话和屏幕共享

### 主要功能

- 🚀 虚拟办公空间：可视化办公环境，显示团队成员位置和状态
- 💬 实时聊天：支持公共频道和私人消息，支持消息翻译
- 📹 视频会议：创建会议室，支持音视频通话、屏幕共享
- 👤 用户管理：用户认证、个人资料管理、头像创建
- 📱 响应式设计：支持桌面设备

## 技术栈

- 前端框架：React + TypeScript
- 桌面应用框架：Electron
- 构建工具：Vite
- UI 组件库：Radix UI
- 样式方案：Tailwind CSS 
- 会议功能：调用腾讯会议 API 实现视频会议功能

- 状态管理：React Hooks
- HTTP 客户端：Axios
- 后端框架：Spring Boot
- 图标库：Lucide React
- 数据库：MySQL
- 数据库 ORM 框架：Spring Data JPA
- 认证与授权：JWT Token 基于角色的访问控制
- 安全措施：密码加密存储、HTTPS 传输、CSRF 防护
- 性能优化：异步处理、缓存策略、数据库索引优化
